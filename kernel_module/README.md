# Kernel module

Simplest linux kernel module. It just prints messages while being loaded or unloaded.

To compile, use

```(bash)
make
```

To load, use

```(bash)
make load
```

To unload, use

```(bash)
make unload
```

To test, use

```(bash)
make test
```

To cleanup, use

```(bash)
make clean
```
